3 Question 3

3.1 Combien d’individus ont un score suppérieur à 635 en lecture (variable reading) dans la base de données STAR ?

3.2 Combien d’individus ont 60 ans ou plus (variable age) dans la base de données BES ?

3.3 Combien de districts on voté à plus de 50% pour le Brexit (variable leave) dans la base de données UK_districts ?

3.4 Combien de pays ont un score d’émission lumineuse suppérieur ou égale à 10 (variable light) dans la base de données countries ?

3.5 Combien de district ont 10% ou plus de vote pour le parti pro russe en 2014 (variable pro_russian) dans la base de donnée UA_precincts ?

Code 
library(dplyr)
# Réponse 3.1 
STAR <- STAR %>% 
  mutate(reading635 = ifelse(reading>635, 1, 0))
sum(STAR$reading635)
[1] 550
Code 
# Réponse 3.2
BES <- BES %>% 
  mutate(age60 = ifelse(age>=60, 1, 0))
sum(BES$age60)
[1] 11267
Code 
# Réponse 3.3
UK_districts <- UK_districts %>% 
  mutate(leave50 = ifelse(leave>50, 1, 0))
sum(UK_districts$leave50, na.rm=T)
[1] 263
Code 
# Réponse 3.4
countries <- countries %>% 
  mutate(light10 = ifelse(light>=10, 1, 0))
sum(countries$light10)
[1] 19
Code 
# Réponse 3.5
UA_precincts <- UA_precincts %>% 
  mutate(pro_russian10 = ifelse(pro_russian>=10, 1, 0))
sum(UA_precincts$pro_russian10)
