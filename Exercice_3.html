3 Model 2 données BES

Avec les données BES, faites une régression logistique multivariée avec:

  • Variable dépendente: Vote pour le Brexit (variable leave).

  • Variable indépendente 1: Le niveau d’éducation en cinq en trois catégories (variable educaion, 1-2 = Bas, 3 = Moyen, 4-5 = Haut)

  • Variable indépendante 2: L’age des individus en 4 catégories (variable age, Moins de 25 ans, de 25 à 44 ans, de 45 à 64 ans, 65 ans ou plus).

Code 
# 1 recoder les variables 

BES <- BES %>% 
  mutate(age_cat = case_when(age<25 ~ "<25",
                             age>=25 & age< 45~ "de 25 à 44",
                             age>=45 & age< 65~ "de 45 à 64",
                             age>=65 ~ ">=65"),
         educ_3 = case_when(education<=2 ~ "Bas", 
                            education==3 ~ "Moyen", 
                            education>3 ~ "Haut"))

# 2 régression logistique: 

model_BES <- glm(leave~educ_3 + age_cat, data = BES, family = binomial(link="logit"))
summary(model_BES)

5 Model 4 données countries

Avec les données countries, faites une régression linéaire avec:

  • Variable dépendente: la différence entre le niveau moyen d’émission de lumière pendant la nuit entre 2005 et 2006, et le niveau moyen d’émission de lumière pendant la nuit entre 1992 et 1993 (variable light - variable prior_light.

  • Variable indépendente: Le taux de croissance du PIB entre les années 2005-2006 et les années 1992-1993 (variable gdp/variable prior_gdp).

Code 
countries <- countries %>% 
  mutate(gdp_growth = gdp/prior_gdp,
         light_change = light - prior_light)
model_countries <- lm(light_change~gdp_growth, data = countries)
summary(model_countries)
